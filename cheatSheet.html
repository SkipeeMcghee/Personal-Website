Brian's Sheet of Cheats

Attribute/Element Tag Description
Heading <h1> to <h6> Represents headings of different levels
Paragraph <p> Defines a paragraph of text
Link <a> Creates a hyperlink to another webpage or resource
Image <img> Inserts an image into the document
List <ul> or <ol> Creates an unordered or ordered list of items
Div <div> Defines a division or section in the document
Class class="classname" Assigns a class to an element for styling or JavaScript interaction
ID id="identifier" Specifies a unique identifier for an element
Inline Style style="property: value;" Adds inline CSS styles directly to an element
